# 動物探險大富翁｜一年級108課綱練習遊戲

這是一個可直接打開的線上網頁遊戲原型，符合以下設定：

- 大富翁繞圈擲骰前進模式
- 可選擇 2～4 人進行
- 45 格地圖
- 一顆骰子
- 所有人都抵達終點才完成遊戲
- 四個一年級低年級方向科目：國語文、數學、生活課程、健康與體育
- 每科 20 題，共 80 題
- 6 個動物角色：小兔子🐰、小熊🐻、小貓🐱、小狗🐶、小狐狸🦊、小企鵝🐧
- 內建音效：擲骰、移動、答對、答錯、過關
- 內建朗讀按鈕：可朗讀題目與狀態
- 不使用外部圖片或音效檔，方便上傳到 GitHub Pages / Netlify

## 從 0 開始使用

### 方法 A：先在自己的電腦玩
1. 解壓縮這個資料夾。
2. 找到 `index.html`。
3. 用 Chrome、Edge、Safari 或 Firefox 打開。
4. 選玩家人數、輸入名字、選動物。
5. 按「開始遊戲」。

### 方法 B：變成隨時可點網址玩的線上版（GitHub Pages）
1. 註冊或登入 GitHub。
2. 建立一個新的 repository，例如：`animal-board-game`。
3. 上傳 `index.html` 到 repository 根目錄。
4. 進入 repository 的 Settings。
5. 點 Pages。
6. Source 選 `Deploy from a branch`。
7. Branch 選 `main`，資料夾選 `/root`。
8. 儲存後等待 GitHub 產生網址。
9. 之後就能用 GitHub Pages 網址進入遊戲。

### 方法 C：變成網址版（Netlify）
1. 註冊或登入 Netlify。
2. 建立新網站，選擇手動部署或拖拉上傳。
3. 把含有 `index.html` 的整個資料夾拖進部署區。
4. 部署完成後，Netlify 會給一個網址。
5. 之後可以把網址分享給學生或家長。

## 修改題目
打開 `index.html`，搜尋 `QUESTION_BANK`，即可看到四科題庫。每題格式大致如下：

```js
{"type":"choice","q":"3 + 2 = ?","options":["5","4","6","2"],"answer":"5","explain":"3個再加2個，一共有5個。"}
```

題型支援：
- `choice`：選擇題
- `tf`：是非題
- `input`：輸入題

## 建議教學用法
- 幼兒或一年級學生建議由老師/家長協助朗讀題目。
- 答錯不扣分，重點是讓孩子看懂提示並願意再試。
- 每局可設定 20～40 分鐘，若時間不足，老師可宣布「目前星星最多者為今日學習王」。
