# 恐龍探險大富翁｜一年級108課綱練習遊戲

這是一個可直接打開的線上網頁遊戲原型。本版已改為蛇行方塊路線、修正題目朗讀會讀到非畫面選項的問題、新增輕快循環背景音樂，並把角色改成 6 種恐龍形象圖。符合以下設定：

- 大富翁擲骰前進模式，地圖為 9×5 的蛇行方塊路線
- 可選擇 2～4 人進行
- 45 格蛇行方塊地圖
- 一顆骰子
- 所有人都抵達終點才完成遊戲
- 四個一年級低年級方向科目：國語文、數學、生活課程、健康與體育
- 每科 20 題，共 80 題
- 6 個恐龍角色：小暴龍、小三角龍、小劍龍、小腕龍、小甲龍、小翼龍
- 內建音效：擲骰、移動、答對、答錯、過關
- 內建輕快背景音樂，可用「背景音樂：開 / 關」按鈕控制
- 內建朗讀按鈕：可朗讀畫面上目前顯示的題目與選項
- 恐龍角色圖檔已放在 `assets/dinosaurs/`，背景音樂與音效仍直接寫在 `index.html` 裡，方便上傳到 GitHub Pages / Netlify

## 本版修正

### 1. 題庫總表亂碼
已另外提供三種題庫檔：

- `題庫總表_Excel可讀版.xlsx`：最推薦，用 Excel 直接開，不容易亂碼。
- `題庫總表_UTF8_BOM.csv`：給 Google 試算表、Numbers 或新版 Excel 使用。
- `題庫總表_BIG5.csv`：給較舊的繁體中文 Windows Excel 使用。

如果你打開 CSV 還是亂碼，請優先使用 `題庫總表_Excel可讀版.xlsx`。

### 2. 恐龍角色圖案
本版已將原本動物 emoji 角色改為 6 張恐龍形象圖，包含角色選擇卡、玩家進度小圖與地圖上的棋子圖案。

圖檔位置：

- `assets/dinosaurs/cards/`：角色選擇使用的大圖
- `assets/dinosaurs/tokens/`：地圖棋子與排行榜使用的小圖

### 3. 輕快背景音樂
背景音樂已直接寫進 `index.html` 裡，不需要額外音檔。因為瀏覽器通常不允許網頁自動播放聲音，所以音樂會在玩家點擊「開始遊戲」或「背景音樂」按鈕後播放。

## 從 0 開始使用

### 方法 A：先在自己的電腦玩
1. 解壓縮這個資料夾。
2. 找到 `index.html`。
3. 用 Chrome、Edge、Safari 或 Firefox 打開。
4. 選玩家人數、輸入名字、選恐龍角色。
5. 按「開始遊戲」。
6. 若不想播放背景音樂，按「背景音樂：開 / 關」。

### 方法 B：變成隨時可點網址玩的線上版（GitHub Pages）
1. 註冊或登入 GitHub。
2. 建立一個新的 repository，例如：`dinosaur-board-game`。
3. 上傳整個資料夾內容到 repository 根目錄，包含 `index.html`、`assets/`、題庫檔與 README。
4. 進入 repository 的 Settings。
5. 點 Pages。
6. Source 選 `Deploy from a branch`。
7. Branch 選 `main`，資料夾選 `/root`。
8. 儲存後等待 GitHub 產生網址。
9. 之後就能用 GitHub Pages 網址進入遊戲。

### 方法 C：變成網址版（Netlify）
1. 註冊或登入 Netlify。
2. 建立新網站，選擇手動部署或拖拉上傳。
3. 把含有 `index.html` 與 `assets/` 的整個資料夾拖進部署區。
4. 部署完成後，Netlify 會給一個網址。
5. 之後可以把網址分享給學生或家長。

## 修改題目
打開 `index.html`，搜尋 `QUESTION_BANK`，即可看到四科題庫。每題格式大致如下：

```js
{"type":"choice","q":"3 + 2 = ?","options":["5","4","6","2"],"answer":"5","explain":"3個再加2個，一共有5個。"}
```

題型支援：
- `choice`：選擇題
- `tf`：是非題
- `input`：輸入題

## 建議教學用法
- 幼兒或一年級學生建議由老師/家長協助朗讀題目。
- 答錯不扣分，重點是讓孩子看懂提示並願意再試。
- 背景音樂音量已設定偏低，朗讀題目時會自動降低音量。
- 每局可設定 20～40 分鐘，若時間不足，老師可宣布「目前星星最多者為今日學習王」。
